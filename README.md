# House Price Prediction - Week 1

## Objective
Predict house prices using machine learning techniques.

## Dataset
Housing.csv (545 records, 13 features)

## Models Used
- Linear Regression
- Random Forest Regressor

## Results
| Model | R² Score |
|---------|---------|
| Linear Regression | 0.653 |
| Random Forest | 0.612 |

## Key Features
- Area
- Bathrooms
- Stories
- Air Conditioning
- Preferred Area
